# GNN_fraud_detection

Dataset used : " Elliptic ++ : https://github.com/git-disl/EllipticPlusPlus.git "

This end-of-year project investigates the practical value of Fraud detection on the Elliptic Bitcoin transaction dataset, a directed graph of transactions with ground-truth illicit labels.
The project implements and compares five architectures; GCN, GraphSAGE, GAT, TAGCN, and temporal GNNs (ATGAT) under a unified experimental framework, with an emphasis on not just what the models predict, but why, through explainability analysis.
